## Diretrizes de Engenharia e Padrões Operacionais

* **Arquitetura de Software Consciente (End-to-End):** Minha experiência com camadas de infraestrutura de redes e sistemas operacionais orienta a escrita do código fonte. Hardening de ambientes, prevenção de gargalos de I/O, otimização de consumo de banda e minimização de latência são premissas adotadas desde o desenho inicial do software.
* **Persistência de Dados e Segurança:** Aplicação rigorosa de engenharia de dados em bancos de dados relacionais. Foco em modelagem lógica e física, integridade referencial, normalização estrutural (1NF, 2NF, 3NF), sanitização avançada de inputs e políticas rígidas de segurança e preservação de dados.
* **Governança de TI e Fluxos Ágeis:** Gestão do ciclo de vida do desenvolvimento de software baseada em frameworks ágeis (Scrum e Kanban). Mapeamento claro de requisitos, rastreabilidade de bugs, documentação de arquitetura técnica em repositórios e versionamento através de fluxos de trabalho eficientes (Git Flow / GitHub Flow).

## Portfólio de Engenharia Aplicada

### Sistemas Embarcados & Integração Mobile Edge
* **Desenvolvimento de Ecossistemas de Monitoramento:** Concepção e arquitetura de soluções integradas ponta a ponta envolvendo hardware dedicado e dispositivos móveis. Implementação de firmware para microcontroladores com comunicação assíncrona, tratamento de concorrência e aquisição de telemetria em tempo real.
* **Interface e Experiência Mobile:** Construção do cliente mobile utilizando React Native e Expo, aplicando conceitos avançados de UI/UX para ambientes críticos sob estresse, garantindo renderização de alta performance e consumo eficiente de bateria.

### Engenharia de Dados & Aplicações Corporativas
* **Microsserviços e APIs de Alta Disponibilidade:** Estruturação de backends robustos em Python e Node.js para agregação, processamento e distribuição de dados estruturados. Implementação de rotas protegidas e controle estrito de acessos.
* **Plataformas de Analytics e Tomada de Decisão:** Desenvolvimento de aplicações de análise preditiva, inteligência de dados e automação de relatórios, focadas no fornecimento de métricas estratégicas detalhadas para stackholders e diretores corporativos.

### Infraestrutura Autónoma & Servidores Locais
* **Virtualização e Redes Domésticas/Empresariais:** Reposicionamento e conversão de hardware legado em servidores utilitários baseados em ambientes Linux. Configuração de roteamento de rede, DNS local, firewalls internos e esteiras de automação de serviços em nível de sistema operacional.
